================================
ACCENTURE MAINFRAME AI ASSISTANT
    PORTABLE APPLICATION
================================

VERSION: 2.0.0 STANDALONE

REQUIREMENTS:
- Python 3.x
- NPX (Node Package Manager)

HOW TO RUN:
-----------
Windows: Double-click run-electron.bat
Linux/WSL: ./run-electron.sh

FEATURES:
- Desktop application (Electron)
- Real SQLite database
- Zero installation required
- Works offline

FILES:
- main.js - Electron main process
- kb-assistant.db - Database with 13 records
- scripts/ - Python backend server
- Accenture-Mainframe-AI-Assistant-Integrated.html - UI

The application will:
1. Start backend server (port 3001)
2. Open desktop window with UI
3. Connect to real database

Close window to stop all services.
